# pol-tech-nexus

The repository contains Python scripts from a Master's thesis about the Nexus between technological development and political engagement of university students, with political efficacy as a mediating variable.

---

## Purpose

This is an archive of the Python code used to perform statistical analyses of the thesis. The code is linked in the thesis using QR codes, so that the reader can see the code used to generate each table and figure.

The data, statistical results, tables and figures are shown in the body of the thesis - this is simply a code repository for transparency and verification.
