# pol-tech-nexus

Code for a Master's thesis on the Nexus Between Technological Advancement
and Political Engagement: A Quantitative Study Among University Students
in Dhaka, Bangladesh.

Each Python script in this repository corresponds to one specific table
or figure in the thesis. Every script reads `data.csv` (held locally,
not uploaded for ethics reasons) and prints or saves the output reported
in the relevant chapter.
